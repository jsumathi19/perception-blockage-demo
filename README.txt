# Perception Blockage Detection - Demo

This repository demonstrates an "Everything-as-Code" approach for an ADAS perception feature.

## Scope
Camera blockage detection using:
- Blur detection
- Contrast analysis

## Structure
- requirements → system & software requirements
- architecture → diagrams (PlantUML)
- design → module-level logic
- code → implementation
- validation → KPI definitions & scripts

## Traceability
Each requirement is linked to:
- design
- code
- KPI validation
